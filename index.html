Los 3 pilares de CSS

Cascada

CSS significa "Hojas de Estilo en Cascada", refiriéndose al orden en que se aplican los estilos:

  1. Estilos en línea
  2. Estilos internos
  3. Estilos externos
  4. Estilos del navegador

Herencia

Los estilos de un elemento "padre" se transfieren a sus "hijos", a menos que se sobrescriban.

Especificidad

Los selectores tienen un nivel de prioridad: ID > Clase > Elemento.
